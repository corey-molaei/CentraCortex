# CentraCortex Frontend

## Stack
- React 19
- Tailwind CSS v4 (`@tailwindcss/vite`)
- Vite 7
- TypeScript

## TailAdmin Integration
- TailAdmin free template reference is vendored under `vendor/tailadmin`.
- Runtime shell and components are implemented in:
  - `src/layout/*`
  - `src/components/ui/*`
- Existing route paths and backend API contracts are unchanged.

## Commands
- `npm install`
- `npm run dev`
- `npm run lint`
- `npm run test -- --run`
- `npm run build`
